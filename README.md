# ♟️ Simulador de Movimentação de Xadrez (C)

Este projeto é um simulador de lógica desenvolvido em **linguagem C**. O programa permite ao utilizador escolher peças clássicas do xadrez e visualizar a sua movimentação no tabuleiro através do terminal, respeitando as direções específicas de cada peça.

## 🚀 Funcionalidades

O simulador cobre a lógica de três peças principais:
- **Torre**: Movimentação ortogonal (Cima, Baixo, Esquerda, Direita).
- **Bispo**: Movimentação diagonal (Cima-Direita, Cima-Esquerda, Baixo-Direita, Baixo-Esquerda).
- **Rainha**: Movimentação combinada (Ortogonal e Diagonal).

### Destaques:
- **Validação de Entrada**: O utilizador pode mover entre 1 a 5 casas (regra customizável).
- **Interatividade**: Menu interativo para escolha de peça e direção via teclado.
- **Feedback Visual**: Exibição passo a passo de cada casa percorrida pela peça.

## 🛠️ Tecnologias Utilizadas

- **Linguagem C**: Base do projeto.
- **Bibliotecas Standard**: `stdio.h` para entrada/saída de dados e `stdlib.h` para funções de controlo de sistema.
